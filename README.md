#iframework

一个轻量级的PHP web开发框架(从零开始开发一个PHP WEB框架)

##开发流程

第一步：了解框架运行流程，确定项目目录结构

第二步：开发应用入口文件和框架入口文件 

第三步：开发框架核心(配置类、路由类、控制器类、模型类、视图类)

第四步：开发应用

第五步：完善框架(添加日志、Sesson等)
